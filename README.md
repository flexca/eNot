
# Description

**Encoding Notations** (**eNot**) is a JSON-based templating engine that serializes data into binary 
**TLV** (Tag-Length-Value) formats, such as **ASN.1** or **BER-TLV**.

The primary goal of **eNot** is to transform JSON templates, featuring placeholders and control structures like loops, 
conditions and functions, into precise binary outputs.
